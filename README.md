# YouTalk — блог о психологии

Верстка блога для компании YouTalk.

## Страницы

- `index.html` — главная страница блога
- `article.html` — страница статьи

## Технологии

- HTML5 (семантическая вёрстка)
- CSS3 (Flexbox, Grid, адаптив)
- БЭМ
- SVG-иконки
- JavaScript (базовый)

## Адаптив

- Desktop (от 1200px)
- Tablet (768px – 1024px)
- Mobile (до 767px)
